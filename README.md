# ⚔️ MTPQI Memory Game — 14 Reinos

🎮 **Jogo educativo interativo** para aprender **Métodos e Técnicas de Pesquisa Quantitativa I (MTPQI)** — Universidade Nove de Julho (UNINOVE).

---

## 🎯 Sobre o Projeto

Jogo da memória **pergunta-resposta multilíngue** com **14 reinos temáticos** em Português, English e 中文.

**Paradigma**: New Statistics (Cumming, 2012) — ênfase em **Effect Sizes, Intervalos de Confiança, e Replicação**.

## 🎮 Como Jogar

1. **Selecione um reino** (🌍 Básico / ⚖️ Intermediário / 👑 Avançado)
2. **Escolha a dificuldade** (Fácil: 4 pares | Normal: 6 pares | Difícil: 8 pares)
3. **Clique em "Começar Jogo"**
4. **Vire as cartas** e encontre os pares pergunta-resposta!
